# LR_1
1. Git - це набір утиліт для відстежування змін у файлах та порівняння, аналізування і відкату бо дільш старої версії вашого проекту.
2. commited, modified, staged.
3. Git add використовується для додавання файлів до репозиторію
4. Git status використовується для перевірки на доданість файлів
5. Git commit використовується для фіксації стану сховища
6. Git init використовується для ініціалізації проекту у папці
7. В коментаріях комітів необхідно писати зміни в репозиторії
8. Щоб переглянути список комітів git config --list
9. Git checkout використовуэться дял переміщення між вітками
